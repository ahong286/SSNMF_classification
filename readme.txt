This code is for our paper "Exploring Structural Consistency in Graph Regularized Joint Spectral-Spatial Sparse Coding for Hyperspectral Image Classification". 

If you use this code, please kindly cite our paper:

Changhong Liu, Jun Zhou, Jie Liang, Yuntao Qian, Hanxi Li,Yongsheng Gao. Exploring Structural Consistency in Graph Regularized Joint Spectral-Spatial Sparse Coding for Hyperspectral Image Classification. IEEE Journal of selected topics in applied earth observations and remote sensing, 2017,10(3): 1151 - 1164.

you can run this code by the following procedures.
(1)TestClassifybySC_CS_Indian_WholeTrain.m: test SC_CS method in the IndianPines dataset.

(2)TestClassifybySC_CS_Pavia_WholeTrain.m: test SC_CS method in he PaviaU dataset.

(3)TestClassifybyGraphSC_CS_Indian_WholeTrain.m: test GSC-CS method in the IndianPines dataset.

(4)TestClassifybyGraphSC_CS_Pavia_WholeTrain.m: test GSC-CS method in the PaviaU dataset.

(5)TestClassifybySC_CS_Indian_WholeTrain_gabor.m: test the GSC-CS_G method in the indiapines dataset.

(6)TestClassifybyGraphSC_CS_Indian_WholeTrain.m: test GSC_CS method in the indianpines dataset.

(7)TestClassifybyGraphSC_CS_Indian_WholeTrain_Gabor.m: test the GSC-CS-G: Test the GSC-CS-G method with the Gabor features in the indianpine.

(8)TestClassifybyGraphSC_CS_Pavia_WholeTrain_Gabor.m: test the GSC-CS-G: Test the GSC-CS-G method with the Gabor features in the Pavia.
